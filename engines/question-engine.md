# MN-ENG-002 - Question Engine

Status: Draft
Version: 0.1.0

This document defines how Med Nykuto builds training questions.

Main rules:

One question tests one learning goal.

Each question needs a prompt, an answer, a correction and a short explanation.

Questions should be useful for learning and exam preparation.

Repeated low value questions should be avoided.

When answer choices are used, choices should be coherent with the question.

This draft will be expanded later.
