# Med Nykuto Core

Med Nykuto Core est le référentiel central du projet Med Nykuto.

Ce dépôt sert à documenter les règles, l'architecture, la pédagogie, les moteurs d'apprentissage, les standards médicaux, les modèles de questions et les procédures de validation.

## Rôle du dépôt

- Définir la logique pédagogique.
- Définir la structure des modules médicaux.
- Définir les règles de création des questions.
- Définir les standards de qualité.
- Garder les décisions importantes du projet.

## Relation avec le site

- `med-nykuto-core` contient les règles et la méthode.
- `med-nykuto` contient le site web.

Le site applique les règles du Core, mais le Core ne doit pas contenir le code principal du site.
